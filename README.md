# Arduino-IR_Truck
Program coded for a Nano board to build a small IR controlled truck for the kids. Uses an I/O expansion board, a matrix LED display and an EZMP3 player. Still a work in progress for the MP3 part.

## Overview
This project is an Arduino-based that I made to build a small IR controlled truck for the kids. It has a full set of light (gyrophare, day lights, brake lights and flashers in the front and in the back), an MP3 player with speakers, a matrix LED display and an I/O expansion board.

## Features
- Basic control logic  
- Commented and structured code for readability

## Skills Highlighted
- Programming in C/C++ for microcontrollers
- Basic electronics principles

## Usage
1. Open `IR_Truck_v2020-11-09.ino` in Arduino IDE
2. Upload to the Arduino board
3. Connect the input and output as per requirements
4. Test the program

## Notes
- Written in 2020 before my formal IT studies; shows structured code and practical electronics knowledge. Also note that no functions were used and that the core of the code is based around several IF structures.
